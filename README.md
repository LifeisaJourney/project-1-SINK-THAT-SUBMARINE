# project-1-SINK-THAT-SUBMARINE
An interactive JavaScript game of a submarine that you have to shoot down.




### Rules
1) Just have fun!
2) Shoot down the submarine!
3) Once you have clicked onto the grid you will either HIT or MISS! You will miss more often than hit.
4) If you MISS, keep trying!
5) Once you HIT! An alert will come up to tell you that you have won!
